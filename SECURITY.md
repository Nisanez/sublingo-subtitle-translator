# Security Policy

SubLingo is a local desktop app. It reads the `.srt` files you open and sends their
text to the Ollama server you configure (by default `http://localhost:11434`). It does not
talk to any other network service, and it ships no secrets.

## Reporting a vulnerability

Please open a **private** security advisory on the GitHub repository, or open a regular
issue if it is low risk. Include:

- A description of the issue and its impact.
- Steps to reproduce.
- Your OS, Node version, and app version.

We'll acknowledge reports as soon as we can. Thanks for helping keep the project safe.
